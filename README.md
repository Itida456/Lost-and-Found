# Lost and Found Tracking System

## Overview

This is a full-stack web application designed to streamline the management of lost and found items. It provides an intuitive interface for reporting lost or found items and helps users easily claim their belongings.

## Features

- User-friendly front-end forms for reporting lost and found items.
- Backend built with Python (Flask) connected to a MySQL database for efficient data storage and retrieval.
- Item tracking and management with claim functionality.
- Responsive design for accessibility on various devices.

## Technologies Used

- Frontend: HTML, CSS
- Backend: Python (Flask)
- Database: MySQL
